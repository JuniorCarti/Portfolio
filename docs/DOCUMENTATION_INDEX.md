## 🎬 Animation Classes at a Glance

```
scroll-fade-in       → Pure fade in (1.2s)
scroll-slide-up      → Slide up + fade (1.4s) ⭐ MAIN
scroll-blur-in       → Blur to sharp (1.6s) 💎 LUXURY
scroll-scale-in      → Scale + fade (1.4s)
scroll-stagger-1/2/3/4 → Staggered reveals (with delays)
```
