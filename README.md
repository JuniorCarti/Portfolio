# Modern Portfolio Website

A stunning, modern portfolio website featuring glassmorphism design, smooth animations, and a professional aesthetic. Built with pure HTML, CSS, and JavaScript.

## ✨ Features

### Design
- **Glassmorphism UI**: Modern glassmorphic design with backdrop blur effects
- **Dark/Light Mode**: Toggle between dark and light themes with persistent preference
- **Animated Background**: Dynamic gradient orbs that create a mesmerizing background effect
- **Smooth Animations**: Scroll-triggered animations and micro-interactions throughout
- **Responsive Design**: Fully responsive layout that works on all devices

### Interactive Elements
- **Typing Effect**: Dynamic typing animation in the hero section
- **Scroll Animations**: Elements fade in as you scroll down the page
- **Progress Bars**: Animated skill progress bars with scroll triggers
- **Counter Animation**: Animated statistics counters
- **Smooth Scrolling**: Smooth scroll navigation between sections
- **Mobile Menu**: Elegant mobile navigation menu

### Sections
- **Hero Section**: Eye-catching introduction with animated badges
- **About Section**: Personal information with animated statistics
- **Skills Section**: Interactive skill cards with progress indicators
- **Education Section**: Timeline-based education history
- **Interests Section**: Cards showcasing areas of interest
- **Projects Section**: Project showcase with hover effects
- **Contact Section**: Contact form with validation

## 🚀 Technologies Used

- **HTML5**: Semantic markup for accessibility
- **CSS3**: 
  - CSS Variables for theming
  - Flexbox & Grid for layouts
  - Custom animations and transitions
  - Glassmorphism effects
  - Gradient backgrounds
- **JavaScript (ES6+)**:
  - Intersection Observer API for scroll animations
  - Local Storage for theme persistence
  - Event delegation
  - Modern async patterns

## 🛠️ Customization

### Changing Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary: #6366f1;
    --secondary: #8b5cf6;
    --accent: #ec4899;
    /* ... */
}
```

### Adding Sections
1. Add HTML structure in `index.html`
2. Add corresponding styles in `css/style.css`
3. Add any JavaScript functionality in `js/script.js`

### Modifying Animations
Animation timings and effects can be adjusted in the CSS file. Look for:
- `@keyframes` definitions
- `transition` properties
- `animation` properties
