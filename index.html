Alice Health

A comprehensive pregnancy monitoring Android application that provides 24-hour support for expectant mothers. The app features personalized health tracking, appointment scheduling, emergency alerts, and educational resources.

Key Features:

  • Real-time health monitoring with wearable device integration
  • Personalized nutrition and exercise plans
  • Instant connection with healthcare providers
  • Educational content tailored to pregnancy stages
  • Appointment scheduling with reminders
  • Emergency alert system with GPS location

Technologies Used:

Java, XML, Firebase, Android SDK, Google Fit API, Retrofit, Room Database

Impact:

The app has helped thousands of expectant mothers track their pregnancy health, resulting in better prenatal care and reduced complications.

Shopfinity (Available on Play Store)

An e-commerce platform specifically designed for bookshops to manage sales, inventory, and bulk orders efficiently. The system includes analytics dashboards for tracking sales performance and customer behavior.

Key Features:

  • Inventory management with barcode scanning
  • Multi-user access with role-based permissions
  • Bulk order processing system
  • Sales analytics and reporting
  • Customer management and loyalty programs
  • Integration with payment gateways

Technologies Used:

Java, Firebase, Android SDK, Google Books API, ML Kit (for barcode scanning), Stripe API

Impact:

Shopfinity has been adopted by over 50 bookshops, helping them increase operational efficiency by 40% and improve sales tracking accuracy.

Banking Management App

A secure mobile banking application that provides multi-user access and transaction capabilities. The app implements advanced security measures including biometric authentication and end-to-end encryption.

Key Features:

  • Budgeting and financial planning tools
  • Fraud detection and alert system

Technologies Used:

Java, SQLite, Android SDK, Encryption Libraries, Biometric API, Retrofit, Room Database

Security Measures:

  • End-to-end encryption for all transactions

  • Regular security audits and penetration testing
