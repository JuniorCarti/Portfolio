## 🔧 Quick Customization

### Change Animation Speed
```css
/* In style.css */
.scroll-slide-up {
    animation: scrollSlideUp 1.4s cubic-bezier(...) forwards;
    /* Change 1.4s to 0.8s (faster) or 2s (slower) */
}
```

### Change Animation Distance
```css
@keyframes scrollSlideUp {
    from {
        transform: translateY(40px); /* Change to 60px, 80px, etc */
    }
}
```

### Add Animation to Element
```html
<!-- In index.html -->
<div class="scroll-slide-up">Your content</div>
```
